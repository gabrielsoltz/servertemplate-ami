Server Template AMI

Script para estandarizar AMI (Amazon Images). El Sistema operaivo default de Amazon EC2. 

./servertemplate-ami.sh HOSTNAME

Funcionalidades:

1. UPDATE SISTEMA
2. ELIMINAR PAQUETES
3. DESACTIVAR SERVICIOS
4. NTP
5. HOSTNAME
6. CONFIGURACION IP (Setea Fija, la ip actual)
7. MOTD
8. AJUSTES SYSCTL
9. AJUSTES MODPROBE
10. DESACTIVAR IPV6
11. SSHD_CONFIG
	- CREAR GRUPO SSH
	- PASSWORD ROOT
	- CREAR USUARIO ADMIN
	- PASSWORD ADMIN
	- KEYS ADMIN
	- SUDOERS ADMIN
	- BASHRC
12. CLOUDWATCH
13. ELIMINAR USUARIO EC2-USER Y SU CONFIGURACION EN SUDOERS.
14. REBOOT
